# 图片素材说明

请【成员二：美术设计与趣味交互】将图片素材放在这个目录下。

## 需要的图片清单

| 文件名 | 说明 | 建议尺寸 |
|--------|------|---------|
| `skin_default.png` | 默认角色（小绿人） | 60×80px |
| `skin_fox.png` | 狐狸皮肤 | 60×80px |
| `skin_bear.png` | 草莓熊皮肤 | 60×80px |
| `skin_bunny.png` | 彩虹兔子皮肤 | 60×80px |
| `obstacle_stump.png` | 树桩障碍物 | 40×70px |
| `coin.png` | 金币 | 30×30px |
| `bg_sky.png` | 天空背景（可选） | 800×300px |
| `ground.png` | 地面瓦片（可选） | 100×50px |

## 图片格式要求
- 使用 PNG 格式，支持透明背景
- 游戏角色必须有透明背景（方便叠加在背景上）
- 图片大小控制在 100KB 以内

## 免费素材来源
- https://opengameart.org （开源游戏素材）
- https://itch.io/game-assets/free （免费游戏素材）
- 可以自己用 Piskel 等工具绘制像素风格图片
