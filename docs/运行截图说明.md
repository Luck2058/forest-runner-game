2. 游戏跑起来后截图（要看到玩家、障碍物、金币同屏）
3. 顶部 HUD 显示分数、金币、距离

截图要点：
- Canvas 画面中有玩家、至少一个障碍物、至少一枚金币
- 顶部分数不为 0
- 颜色丰富（天空蓝、地面绿、障碍物棕色、金币黄）

## 截图建议

| 建议 | 说明 |
|------|------|
| 浏览器 | 建议使用 Chrome，截图最清晰 |
| 分辨率 | 建议全屏 1920×1080，截图后适当裁剪 |
| 工具 | Windows 系统用 Win+Shift+S 截取区域 |
| 命名 | 按序号命名，如 `01_登录页.png`、`02_注册页.png` |
| 保存位置 | 统一存入 `docs/screenshots/` 文件夹 |
| 数量 | 每个页面至少截 1 张，重要页面（游戏中/游戏结束）截 2 张 |

## 录屏建议（可选）

若答辩时需要演示视频，建议录制以下流程：

1. 访问首页（展示动画效果）
2. 注册账号
3. 登录账号
4. 开始游戏（玩 30 秒以上）
5. 游戏结束，查看成绩
6. 查看排行榜
7. 进入个人中心
8. 进入后台管理

**推荐录屏工具**：
- OBS Studio（免费，支持 MP4 输出）
- Bandicam（轻量，适合桌面录屏）
- Windows 自带：Win+G（游戏录制）
